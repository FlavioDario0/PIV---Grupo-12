# Kineo - Seu Guia Personalizado de Saúde Física

Bem-vindo ao repositório do **Kineo**, projeto desenvolvido como parte da disciplina de **Projeto Integrador V – Engenharia de Software** da Pontifícia Universidade Católica de Campinas (PUC-Campinas).

## 📖 Sobre o Projeto

Embora a busca por um estilo de vida mais saudável tenha crescido, o acesso a orientações físicas verdadeiramente personalizadas ainda é um grande desafio. Planos de treino genéricos frequentemente ignoram características individuais críticas (como idade, peso e limitações físicas), o que frequentemente resulta em frustração, estagnação e, em casos graves, lesões. Por outro lado, o acompanhamento profissional individualizado apresenta um custo elevado, inacessível para muitos.

O **Kineo** surge para preencher essa lacuna de mercado, oferecendo uma solução tecnológica acessível que une:
- 🏋️ **Planejamento físico estruturado e 100% personalizado.**
- 🧠 **Inteligência Artificial** para adaptar os treinos com base em seus dados e progresso reais.
- 📚 **Educação em saúde**, ajudando a filtrar a sobrecarga de informações fragmentadas e confusas disponíveis na internet.

## 🚀 Funcionalidades Principais (Em Desenvolvimento)

- **Geração de Treinos via IA:** Algoritmos inteligentes que recomendam exercícios adaptados à realidade de cada usuário.
- **Acompanhamento de Progresso:** Dashboards modernos e intuitivos para monitorar estatísticas, níveis e evolução semanal.
- **Design de Alta Fidelidade:** Interface limpa, responsiva e com foco total na melhor Experiência do Usuário (UX/UI).

## 🏫 Contexto Acadêmico (Grupo 12)

Este projeto foi desenhado para atender aos requisitos da disciplina de Engenharia de Software (PI-V). Envolve a aplicação prática de metodologias ágeis (Scrum), modelagem de arquitetura, testes de software, e integração direta com a comunidade externa (Curricularização da Extensão) para validação dos requisitos e da IA.
